# Orienteering Web for SPSSA Event

## Requirement

NodeJS v11 or above

## Deploy

clone this repo
```
git clone https://github.com/EtBLh/orienteering.git
```

install dependencies
```
cd client && npm i
cd ../server && npm i
```

build project
```
cd client %% npm build
cd ../server && npm build
```

start server
```
cd server && npm start
```

## Remarks

this app is using port 5000
